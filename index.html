Installation Guide

1. Install OpenClaw

iwr -useb https://openclaw.ai/install.ps1 | iex

2. Create Telegram Bot via @BotFather

3. Add Telegram config with your bot token

4. Start gateway:

openclaw gateway

5. Approve pairing:

openclaw pairing approve telegram <CODE>
